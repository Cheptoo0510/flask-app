Student Report Package

This Python package contains:

1. grades.py
   - Function: calculate_average(grades)
   - Purpose: Returns the average grade from a list of numbers.

2. attendance.py
   - Function: calculate_attendance(attendance_list)
   - Purpose: Calculates the percentage of classes attended (1 = attended, 0 = missed).

3. report.py
   - Purpose: Imports both functions above and generates a report.
   - How to use: Run with `python report.py`. The output will show the average grade and attendance percentage.

Instructions:
- Make sure Python is installed.
- Run the report.py file from terminal or command line.
